Jag har gjort en hemsida gjord för att räkna ut olika värden med hjälp av inbyggda formler som till exempel energi, kraft och relativitet.

Tanken med projektet är att det ska vara enkelt att räkna ut fysikaliska svar utan att behöva knappa på en vanlig miniräknare. Man skriver bara in sina värden, så sköter koden resten.
